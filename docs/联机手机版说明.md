# Booxin 联机手机版（bphone）

手机版是 **联机通知伴侣**，不负责在手机上启动 Minecraft 或加入房间。

## 功能范围

| 功能 | 说明 |
|------|------|
| 注册 / 登录 | 与 PC 启动器共用联机账号 |
| 公共房间 | 只读浏览，在 PC 启动器加入 |
| 好友列表 | 查看好友、处理好友申请 |
| 用户大厅 | 浏览全站用户、发送好友申请 |
| 联机邀请通知 | 轮询 `GET /api/friends`，新邀请弹窗 + 本地推送 |

## 使用的 API

### 认证服务（5005 / bbx）

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/auth/me`
- `POST /api/auth/change-password`（账户页）
- `GET /api/auth/users/lobby`

### 好友服务（5005 / bbx）

- `GET /api/friends`
- `POST /api/friends/request`
- `POST /api/friends/requests/{id}/accept`
- `POST /api/friends/requests/{id}/reject`
- `POST /api/friends/invites/{id}/dismiss`

### 房间服务（5000）

- `GET /api/rooms?isPublic=true`
- `GET /api/rooms/{roomCode}`（详情可选）

## 邀请通知逻辑

1. 登录后每 **45 秒** 请求 `GET /api/friends`
2. 对比 `pendingRoomInvites`，首次登录只建立基线不弹历史邀请
3. 新邀请 → 应用内弹窗 + Expo 本地通知
4. 实际进房请在 **PC Booxin 启动器** 输入房间码

客户端封装：`lib/_core/booxin-api.ts`
